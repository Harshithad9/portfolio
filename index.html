Coffee Shop Management System

Description

A console-based application developed in C to manage daily operations of a coffee shop, including order processing, inventory management, and sales reporting.

Key Features

  • Menu Management: Add/update/delete coffee items and prices
  • Order Processing: Take customer orders and calculate bills
  • Inventory Tracking: Monitor stock levels of coffee beans/milk/supplies
  • Sales Reports: Daily/weekly sales summaries
  • User Authentication: Separate login for staff and managers

Technical Implementation

  • Built using C with file handling for data persistence
  • Modular structure with separate functions for each feature
  • Uses arrays and structures to manage menu items and orders
  • Error handling for invalid inputs

Quiz Quest

Description

QuizQuest is a Python-based interactive quiz game that tests users' knowledge across multiple categories with score tracking and progress saving features.

Core Features

  • Multi-Category Quizzes: Science, History, Pop Culture, etc.
  • Score System: Points for correct answers with time bonuses
  • User Profiles: Save progress and high scores
  • Question Database: 100+ questions in JSON format
  • Difficulty Levels: Easy/Medium/Hard modes

Technical Implementation

  • Built with Python using Tkinter for GUI
  • Question data stored in JSON files
  • Object-oriented design with separate classes for Quiz/User/Question
  • Timer functionality for each question
  • Error handling for user inputs

Learning Outcomes

  • Implemented file handling for data persistence
  • Gained experience with GUI development
  • Applied OOP principles in a practical project
